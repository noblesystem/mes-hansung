Visual Studio에서 ASP.NET MVC 5에 대한 전체 집합 종속성을 'hansung' 프로젝트에 추가했습니다. 

ASP.NET MVC을(를) 사용하도록 설정하려면 프로젝트의 Global.asax.cs 파일을 추가로 변경해야 합니다.

1. 다음 네임스페이스 참조를 추가합니다:

c

2. 코드에 Application_Start 메서드가 아직 정의되어 있지 않으면 다음 메서드를 추가합니다:

    protected void Application_Start()
    {
    }

3. Application_Start 메서드 끝에 다음 줄을 추가합니다:

    AreaRegistration.RegisterAllAreas();
    RouteConfig2.RegisterRoutes(RouteTable.Routes);
    FilterConfig.RegisterGlobalFilters(GlobalFilters.Filters);
    BundleConfig2.RegisterBundles(BundleTable.Bundles);
